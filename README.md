# In Vera Dent 🦷

თბილისური სტომატოლოგიური კლინიკის მრავალგვერდიანი ვებ აპლიკაცია — სერვისების
კატალოგი, ფასები, ონლაინ ჩაწერა და ადმინ პანელი ჩაწერების სანახავად.

## გაშვება

პროექტი იყენებს ES მოდულებს (`<script type="module">`), ამიტომ ფაილების
პირდაპირ `file://`-ით გახსნა არ მუშაობს (ბრაუზერის CORS შეზღუდვა). გაუშვი
ლოკალური სერვერით, მაგ.:

```bash
python3 -m http.server 8000
# შემდეგ გახსენი http://localhost:8000
```

ან VS Code-ის **Live Server** გაფართოებით.

| `services.html` | სერვისების დეტალური აღწერა კატეგორიების ნავიგაციით |

## ტექნოლოგიები

- **HTML5** — სემანტიკური სტრუქტურა, BEM კლასების კონვენცია
- **CSS3** — ერთი გარე ფაილი, CSS custom properties, Flexbox + Grid, mobile-first
  რესპონსიული დიზაინი (480 / 768 / 1024 breakpoint-ები)
- **JavaScript (ES Modules)** — დაყოფილია ერთ-მიზნიან მოდულებად, `fetch`
  გარე API-ებზე `async/await`-ით, `localStorage`, დინამიური DOM

```
├── *.html              # 6 გვერდი, ერთმანეთთან <a href>-ით დაკავშირებული

│   └── main.css        # ყველა სტილი
├── js/
│   ├── main.js         # entry point (<script type="module">)

│   ├── auth.js         # ადმინ პანელის გეითი
│   ├── booking.js      # ჩაწერის ფორმა
│   ├── notify.js       # ელ-ფოსტის შეტყობინება (Web3Forms)
│   ├── data.js         # სერვისების საერთო კატალოგი
│   └── utils.js        # debounce, კალკულატორი, formatDate, showBanner

```

## ძირითადი ფუნქციები

- **ჩაწერის ფორმა** — სრული კლიენტის მხარის ვალიდაცია, `e.preventDefault()`,
  წარმატების/შეცდომის feedback; ჩანაწერი ერთდროულად იგზავნება API-ზე და
  (კონფიგურაციისას) ელ-ფოსტაზე `Promise.allSettled`-ით.
- **ადმინ პანელი** — `fetch`-ით ჩატვირთული ჩაწერები ჩატვირთვის ინდიკატორითა
  და შეცდომის დამუშავებით; debounce-ით აღჭურვილი ძებნა და კატეგორიის ფილტრი
  ინახება `localStorage`-ში.
- **ღირებულების კალკულატორი** — closure-ზე დაფუძნებული (private `Set`),
  არჩევანი ინახება გვერდის გადატვირთვებს შორის.
- **Before/After სლაიდერი** — ერთი `--pos` ცვლადი მართავs კლიპსაც და სახელურსაც.
